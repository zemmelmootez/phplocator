PHPLocator Browser Extension Privacy Policy

Last Updated: August 28, 2025
Effective Date: August 28, 2025

Overview

PHPLocator is a developer tool that enables click-to-code functionality for PHP development. This privacy policy explains our data practices for the PHPLocator browser extension.

Data Collection

PHPLocator does not collect, store, transmit, or share any user data.

Specifically, we do not collect:

  • Personal information (names, emails, addresses)
  • Browsing history or website content
  • User activity or behavior data
  • Location data or device information
  • Authentication credentials
  • Financial or payment information
  • Any form of personally identifiable information

How PHPLocator Works

The extension operates entirely locally on your device:

  1. Detects Alt+click events on HTML elements
  2. Reads local data attributes (data-php-file, data-php-line) from the HTML
  3. Opens the corresponding file in VS Code using local protocol URLs
  4. All processing happens locally - nothing is sent to external servers

Permissions Usage

The extension requests the following permissions:

  • activeTab: Required to inject scripts that detect clicks and read data attributes on the current tab
  • Host Permissions (all sites): Required to work on any website during PHP development (localhost, staging, production)

These permissions are used solely for the extension's core functionality and not for data collection.

Third-Party Services

PHPLocator does not integrate with, send data to, or receive data from any third-party services, analytics platforms, or external APIs.

Data Storage

The extension does not store any data locally or remotely. It operates purely as a click-to-code interface without persistence.

Changes to This Policy

Any changes to this privacy policy will be posted on this page with an updated "Last Updated" date. Since we don't collect data, changes would only be made for clarity or legal compliance.

Open Source

PHPLocator is open source software. You can review the complete source code at https://github.com/zemmelmootez/phplocator to verify our privacy practices.
